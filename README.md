# Pong Game

A simple Pong game built with HTML, CSS, and JavaScript.

## How to Play

- **Left Paddle:** W (up), S (down)
- **Right Paddle:** Up/Down arrow keys

## Run Locally

Open `index.html` in your browser, or  
Enable GitHub Pages in repository settings to play online!
